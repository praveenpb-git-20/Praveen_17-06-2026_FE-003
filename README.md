# FlowTrack 360

## Overview
FlowTrack 360 is a reusable component library developed using React.

## Features

- Reusable Buttons
- Reusable Cards
- Reusable Tables
- Fully Responsive Design
- Modern UI

## Technologies Used

- React JS
- CSS3
- JavaScript

## Installation

```bash
npm install
npm run dev
```

## Components

### Button
Supports:
- Add Employee
- Approve Task 
- Delete Employee

### Card
Displays reusable information cards.

### Table
Displays dynamic tabular data.

## Responsive Design

Works across:
- Desktop
- Tablet
- Mobile
